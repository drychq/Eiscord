\subsection{可用性与可靠性要求}

\begin{enumerate}[itemsep=0pt]
  \item \textbf{服务可用性}：核心 API 服务目标可用性 99.5\%，允许每月不超过 3.6 小时的计划外停机。
  \item \textbf{优雅降级}：当 Redis 不可用时，在线状态查询回退到数据库轮询（降级模式）；当 MinIO 不可用时，文件上传返回 503 并提示稍后重试。
  \item \textbf{数据持久性}：PostgreSQL 主库每日自动备份（pg\_dump），MinIO 文件存储启用 Bucket 版本控制。
  \item \textbf{故障恢复}：API 服务进程崩溃后由 Docker Compose 自动重启（restart: unless-stopped），Mediasoup Worker 崩溃后由 Node.js 子进程管理模块自动拉起。
  \item \textbf{消息可靠性}：WebSocket 断连重试采用指数退避（1s/2s/4s/8s/16s），重连后自动同步离线期间未读计数。
\end{enumerate}

\subsection{安全与隐私要求}

\begin{enumerate}[itemsep=0pt]
  \item \textbf{认证安全}：密码采用 bcrypt（cost=12）哈希存储；Access Token 有效期 15 分钟，Refresh Token 有效期 7 天，Refresh Token 轮换使用后旧令牌即时失效。
  \item \textbf{传输安全}：HTTPS（TLS 1.3）用于所有 HTTP 流量；WSS（WebSocket Secure）用于实时通信；WebRTC 媒体流强制 DTLS-SRTP 加密。
  \item \textbf{输入校验}：所有 API 入参由 Zod Schema + NestJS ValidationPipe 双重校验，防止 SQL 注入（Prisma 参数化查询）和 XSS（前端输出转义）。
  \item \textbf{速率限制}：全局 ThrottlerGuard：登录 5次/分钟/IP，消息发送 30次/分钟/用户，文件上传 10次/分钟/用户。超出限制返回 429 状态码。
  \item \textbf{CORS 策略}：仅允许前端域名（web.eiscord.local）通过配置白名单访问 API，禁止跨域 WebSocket。
  \item \textbf{敏感信息保护}：日志中不记录密码明文和 Token 全文；JWT payload 仅包含 userId，不包含密码或角色信息。
\end{enumerate}

\subsection{可维护性与可扩展性要求}

\begin{enumerate}[itemsep=0pt]
  \item \textbf{模块化架构}：NestJS 按功能领域拆分为 14 个独立模块，每个模块包含 controller / service / gateway，支持按需加载和独立测试。
  \item \textbf{配置管理}：所有环境相关配置通过环境变量注入（.env），不硬编码；区分 development / test / production 三套环境。
  \item \textbf{日志与监控}：API 使用 NestJS Logger，关键操作（登录、消息发送、权限变更）记录结构化日志；留有 Prometheus metrics 接入点。
  \item \textbf{水平扩展}：API 服务为无状态设计（Session 存于 Redis），可通过增加容器实例实现水平扩展；Socket.IO 使用 Redis Adapter 实现多节点广播。
  \item \textbf{数据库迁移}：使用 Prisma Migrate 管理所有 DDL 变更，迁移文件纳入版本控制；支持 migrate deploy（生产）和 migrate dev（开发）两种模式。
\end{enumerate}

\subsection{易用性与兼容性要求}

\begin{enumerate}[itemsep=0pt]
  \item \textbf{浏览器兼容}：支持 Chrome 120+、Firefox 128+、Edge 120+（均需支持 WebRTC 和 WebSocket）。
  \item \textbf{响应式设计}：Web 前端基于 Tailwind CSS 实现响应式布局，适配桌面端（1024px+）和平板端（768px--1023px）。
  \item \textbf{国际化准备}：前端文案使用 i18n 键值方式管理，当前仅提供中文简体（zh-CN），架构上预留英文（en-US）扩展能力。
  \item \textbf{无障碍}：语义化 HTML 标签，关键交互元素提供 aria-label；色彩对比度符合 WCAG AA 级标准。
\end{enumerate}
